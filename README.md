# 中国象棋 AI 引擎

纯 JavaScript 实现的中国象棋 AI 引擎，单文件 HTML，浏览器直接打开即可人机对战。

## 技术特性

- **Alpha-Beta 剪枝** + 迭代加深搜索
- **置换表** (Zobrist 64-bit 哈希)
- **静止搜索** (Quiescence Search)
- **走法排序**: MVV-LVA + 杀手走法 + 历史启发表
- **评估函数**: 棋子价值 + 位置价值表 (PST) + 机动性 + 将军检测 + 残局调整
- **完整规则**: 蹩马脚、塞象眼、炮架、飞将、长将检测、困毙/将死判定

## 使用方式

```bash
# 直接用浏览器打开
open index.html
```

## 难度档位

| 档位 | 搜索深度 | 说明 |
|------|---------|------|
| 初级 | 1 | 随机水平 |
| 中级 | 3 | 基础战术 |
| 高级 | 4 | 业余水平 (~1s) |
| 大师 | 5 | 较强业余 (~3-5s) |
| 特级 | 6 | 强业余 (~8-15s) |

## 操作

- 点击棋子选中 → 绿色圆点显示合法走法 → 点击目标位置走棋
- 键盘快捷键: `N` 新局 `U` 悔棋 `S` 换边
- 可选人机对战 (执红/执黑)
